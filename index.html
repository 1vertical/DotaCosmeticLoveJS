Table Design Changes

Added

  • Added new columns. (See Notes to see what they are and what they represent.)

Changed

  • Reordered columns to make comparisons easier.
  • Data source changed from gamepedia.com to fandom.com (former URL reroutes to the fandom.com URL).

Removed

  • "Broken down?" column is removed. All heroes are currently released. For the uninitiated, this column tracked which heroes were ready/partially ready/not ready for getting new equipment. Check the past publication for the list broken down status of all the heroes.

Notes

Column Explanations

  • Hero ~ The hero in question.
  • Slots ~ The total amount of slots for the character.
  • Past Sets ~ The past total of published sets.
  • Current Sets ~ The current total of published sets.
  • Set Difference ~ The difference between past sets and current sets.
  • Past Items ~ The past total of published items.
  • Current Items ~ The current total of published items.
  • Item Difference ~ The difference between past items and current items.
  • Average Factor ~ As suggested by u/FakeTherapist - The average items per number of slots is included to minimize bias. Formula: Total Items/Slots.
  • Arcanas ~ The hero has an arcanas or not.
  • Personas ~ The hero has a personas or not.
  • Taunts ~ The hero has taunt animations or not.
  • Kinetic Gems ~ The hero has kinetic gems or not.
  • Voices ~ The hero has other voices or not.
  • Release Date ~ The release date of the hero. (In Dota 2's client.) Also, poor IO. :(

Trophy Glyphs

  • Indicates their previous position on the previous analysis of being loved. For example: 1st = Most loved. 99th is the least. X is when the hero didn't exist.

Totals are Variable

  • The totals here is an approximation - I count by hand and don't web crawl the source site and it is very possible that items were added after I was done counting.

Variable Items per Set

  • Sets don't necessarily include items made for all slots. Examples: Single items, Arcanas, early lifetime sets - typically a set with 3 items instead of an item for all slots, etc.

Slots vs. Total Inflation

  • Each hero has their own amount of slots ~ this can inflate/deflate the ratio leading towards a biased total of items. The Average Factor column is added for an inflation factor for sets.
  • Variable Slots
    • Some heroes' (like tinker for example) slots differ from the website source total and workshop requirements. The maximum number is used.
  • Slots Column Updated ~ The slots column was updated to the current slot counts. You will notice that some heroes' slots are also updated from the old count.
  • No Past Workshop Sets Column ~ There is no "Workshop Sets (Past)" column because I didn't record it back in the day.
  • Weapon Packs ~ Weapon packs are considered sets here and added in the Set column for heroes that have them.
  • Arcanas
    • Some arcanas are either a set of items or a single item.
  • Duplicate Items

Duplicate and Gold Items

  • Some heroes have duplicates of the same item (like a gold version of an item for example).
  • These items are part of the total count.
  • Not all heroes have duplicates though which is also a factor of inflation.
  • Gold items are considered separate items from their standard counterparts.
